# FusionFinder聚合搜索平台前端

## 技术栈
- Vue3
- Ant Design Vue 组件库
- 页面状态同步

## 项目简介
基于Vue3 + SpringBoot +Elastic Stack 的一站式聚合搜索平台。

对用户来说，使用该平台，可以在同一个页面集中搜索出不同来源、不同类型的内容，提升用户的检索效率和搜索体验。

对企业来说，当企业内部又多个项目数据都存在搜索需求时，无需针对每个项目单独开发搜索功能，可以直接将各项目的数据源接入搜索中台，从而提升开发效率、降低系统维护成本。
